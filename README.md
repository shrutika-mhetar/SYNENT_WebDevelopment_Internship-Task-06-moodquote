# ✨ MoodQuote – Interactive Quote Generator

A beautifully designed quote application that delivers inspirational quotes through API integration and provides a personalized experience.

## 📌 Project Overview
MoodQuote allows users to generate, save, copy, and manage quotes while exploring a modern interactive interface.

## 🚀 Features
✔ API Integration  
✔ Dynamic Quote Generation  
✔ Copy Quote Functionality  
✔ Save Favorite Quotes  
✔ Delete Saved Quotes  
✔ localStorage Support  
✔ Responsive Premium UI  

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- Fetch API

## 📱 Responsive
Built for both Desktop and Mobile experiences.

## 🎯 Learning Outcome
Strengthened API handling, asynchronous JavaScript, UI development, and local storage concepts.
